.. start

=======================================
start
=======================================

.. code-block:: bash

   使い方: docker start [オプション] コンテナ [コンテナ...]
   
   １つまたは複数のコンテナを起動する
   
     -a, --attach               STDOUT、STDERR にアタッチする
     --detach-keys              コンテナのデタッチに使うエスケープ・キー・シーケンスを設定
     --help                     使い方の表示
     -i, --interactive          コンテナの STDIN にアタッチ
