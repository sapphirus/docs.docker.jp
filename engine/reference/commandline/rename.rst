.. rename

=======================================
rename
=======================================

.. code-block:: bash

   使い方: docker rename [オプション] 古い名前 新しい名前
   
   コンテナの名前を変更
   
     --help          使い方の表示

.. The docker rename command allows the container to be renamed to a different name

``docker rename`` コマンドは、コンテナ名を別の名前にします。
