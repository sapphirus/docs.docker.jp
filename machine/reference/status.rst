=======================================
status
=======================================

.. code-block:: bash

   使い方: docker-machine status [引数...]
   
   マシンの状態を取得
   
   説明:
      引数はマシン名

.. Get the status of a machine.

マシンのステータス（状態）を取得します。

.. code-block:: bash

   $ docker-machine status dev
   Running
