=======================================
start
=======================================

.. code-block:: bash

   使い方: docker-machine start [引数...]
   
   マシンを起動
   
   説明:
      引数は１つまたは複数のマシン名

マシンを起動します。

.. code-block:: bash

   $ docker-machine start dev
   Starting VM...
